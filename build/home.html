Ryan's Inventory

Welcome to Ryan's personal inventory system

Please select an operation from the following form, then provide parameters(s) when appropriate

These are the available functions:

  • Print Inventory - Takes a parameter in the form "x,y" and will display items in inventory in range from x to y
    Note: If the parameter is left blnak, all available inventory items will be displayed
  • Compute Inventory - Takes no parameters and retuns the total worth of all inventory items
  • Print Category - Takes the name of a category and displays all itmes in inventory associated with the category defined
    Possible Categories: Book, CD, Collectable, HomeGarden, Fashion, Electronics
  • Search Inventory - Takes a search keyword and retuns the all items in inventory containing that keyword
